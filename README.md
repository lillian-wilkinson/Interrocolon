# Interrocolon
Punctuation symbol representing the combination of a question mark and a colon.

<img width="270" height="340" alt="image" src="https://github.com/user-attachments/assets/7bb231f6-7af7-43af-afc8-028811082703" />

<h3>How do I use it?</h3>

* When asking a question about a list
* When something is syntactically introduced by a colon but belongs to a question
* Any time you want to use both a colon and question mark in the same place

For example:

"Which of these do you prefer <img width="10" height="12" alt="image" src="https://github.com/user-attachments/assets/7bb231f6-7af7-43af-afc8-028811082703" />
* Apple
* Banana
* Orange"

<h3>Why is this needed?</h3>

Currently, when introducing lists that belong to a question, there are three options:

* Use a question mark only (most common): "Which of these do you prefer?"
* Use a question mark followed by a colon: "Which of these do you prefer?:"
* Use a colon followed by a question mark: "Which of these do you prefer:?"

All three options are unrefined, as they result in either the list not being introduced properly or double punctuation marks which appear out-of-place. The interrocolon addresses this.

<h3>How do I draw it?</h3>

Start with an ordinary question mark. If it has a square dot at the bottom, replace it with a circle.

Add an additional circle to the question mark, approximately overlaying a colon to it. It should be in line with the bottom dot and approximately in the centre of the question mark's curvature, creating an aesthetically pleasing doughnut shape out of the white space.

<h3>How can I type it?</h3>

There is currently no Unicode symbol for the interrocolon, which would be required for fonts and software to support it.

The Unicode consortium generally requires new symbols to already be in-use before they can be accepted. If you feel that the interrocolon serves a useful purpose, start using it where you can and please do get in touch, as the more evidence I can provide of its use, the more likely a proposal is to suceed.
